## 1.0.0

- Added `pdf-to-image` command (render pages to images).
- Added `image-to-pdf` command (combine images into a PDF).
- Added `watermark` command (text and image watermarks).
- Added `ocr` command (searchable PDF and text extraction, via Tesseract).
- Added `markdown` command (export PDF text to Markdown, via pymupdf4llm).

## 0.1.0

- Added PDF parser module.
- Added PDF metadata extraction.
- Added text extraction utilities.
- Added `info` CLI command.
